# AstroAgents

## Run local env

Export the required environment variables:

```bash
export GOOGLE_API_KEY=''
export GITHUB_REPO=''
export GITHUB_TOKEN=''
```

Run the agents in dev mode:

```bash
go run cmd/all/main.go web api webui
```

Visit [http://localhost:8080/ui/](http://localhost:8080/ui/)
